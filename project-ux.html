WISE TALK

OVERVIEW

The Wise Talk - Audiobook application design project aims to build an intuitive and user-friendly platform that allows users to seamlessly discover, manage, and enjoy audiobooks on their mobile devices. From the initial ideation phase with basic wireframes to the detailed and aesthetic interface design, the project demonstrates a strong focus on user experience and the core functionality of an audiobook application.
